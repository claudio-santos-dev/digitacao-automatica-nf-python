# Projeto de Digitação Automática de NFs Fictícias

## Sobre o projeto
Este projeto em Python tem como objetivo automatizar a digitação de Notas Fiscais fictícias a partir de dados extraídos de uma planilha Excel.

O sistema identifica quais ORs precisam ser digitadas, organiza os dados (produtos, quantidades e clientes) e realiza o processo automaticamente dentro de um sistema web.

---

## Problema que resolve

Processo manual:
- Digitação pode levar de **10 minutos a 1 hora por nota**
- Alto volume de trabalho repetitivo

Com automação:
- Redução significativa de tempo
- Capacidade de processar **múltiplas ORs automaticamente**
- Maior eficiência e menos esforço manual

---

## Fluxo esperado do sistema

1. Ler planilha Excel contendo:
   - ORs
   - Produtos
   - Quantidades
   - Clientes  

2. Processar e organizar os dados  

3. Acessar o sistema web:
   - Realizar login  
   - Navegar até as telas necessárias  

4. Executar a digitação automática:
   - Inserir dados de cada OR  
   - Validar informações  
   - Salvar  

5. Repetir o processo até finalizar todas as ORs
